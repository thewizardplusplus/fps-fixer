# FPS Fixer

## Requirements

- [FFmpeg](https://ffmpeg.org/) >=4.4.2, <5.0.

## Usage

```
$ fps_fixer.bash
```

Environment variables:

- `VIDEO_EXTENSION` &mdash; video file extension (default: `mp4`);
- `FIXED_VIDEO_BASE_PATH` &mdash; base path for fixed videos (default: `./fixed-videos`).
